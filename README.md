## Introduction

This ebook is available online at [http://conversation.bigbinary.com](http://conversation.bigbinary.com).

This book is written using [GitBook](http://www.gitbook.io).

## How to start the server

```shell
brew bundle
gitbook serve ./ebook
gitbook build ./ebook
```
